# EventFlow - Premium Brand Guidelines

## 📋 Overview

EventFlow is a modern, luxurious event management platform designed for discerning users who expect premium quality and cutting-edge technology. The brand identity reflects innovation, sophistication, and digital excellence.

---

## 🎨 Visual Identity

### Core Concept
- **Theme**: Futuristic 3D Flow & Movement
- **Feel**: Premium Tech, Modern SaaS, Luxurious Digital
- **Personality**: Modern, Advanced, Professional, High-Value
- **Energy**: Dynamic, Flowing, Immersive, Innovative

### Logo Variations

#### 1. **Main Logo Version**
- Full branding with dynamic 3D circles rotating around a central core
- Represents flow, energy, and movement
- Use for primary branding, websites, and splash screens
- Includes animated elements with soft glowing effects

#### 2. **Icon-Only Version**
- Standalone "E" with gradient and floating animation
- Strong enough to work as app icon, favicon, or avatar
- Memorable and instantly recognizable
- Use when space is limited or symbol recognition is needed

#### 3. **Dark Background Version**
- Optimized for dark UI dashboards
- Neon rings with glowing effects
- Premium feel on dark interfaces
- Perfect for app backgrounds and dark mode implementations

#### 4. **Transparent Version**
- For overlays and flexible integration
- Conic gradient circles with depth
- Use on various background colors
- Flexible for creative applications

## ✨ Visual Effects

### Animations
- **Rotation**: Smooth 3D rotations (6-20s duration)
- **Float**: Subtle up/down motion (4s duration)
- **Glow**: Pulsing opacity and blur (2-3s duration)
- **Scaling**: Micro-interactions on hover/active states

### Effects
- **Glassmorphism**: Frosted glass with 10-20px blur
- **Neon Glow**: Box-shadow with colored light
- **Depth**: Multiple shadow layers and layering
- **Bloom**: Soft outer glow on interactive elements

### Particles & Motion
- Flowing dots along circular paths
- Rotating concentric circles
- Pulsing borders and light
- Smooth ease-in-out transitions

---

## 🎯 Usage Guidelines

### ✅ Do's
- Use on dark backgrounds for maximum impact
- Apply consistent animations for premium feel
- Maintain proper spacing around logo
- Preserve gradient direction (135° angle)
- Use with complementary colors from palette
- Apply glowing effects for depth

### ❌ Don'ts
- Change logo proportions (maintain aspect ratio)
- Remove gradient effects (use solid colors only if necessary)
- Distort or stretch the logo
- Use childish bright colors
- Remove shadows/glows in core brand applications
- Modify the core circular concept

## 📱 Implementation Sizes

### Web
- **Header Logo**: 40-50px height

### Mobile
- **App Icon**: 192x192px, 512x512px (with safe area)
- **Navbar Logo**: 40-45px height

### Print
- **Logo Mark**: Minimum 2cm
- **Full Logo**: Minimum 4cm height
- **Print Resolution**: 300DPI minimum

## 🎬 Animation Specifications

```

Easing: ease-in-out
Loop: infinite
```

```

Duration: 3 seconds
Easing: ease-in-out
Loop: infinite
```
